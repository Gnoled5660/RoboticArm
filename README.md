# Description
My goal is to build a robotic arm, that can be controlled over toggle switches. The robotic arm has four axis consisting of four powerful servos. Another servo is used to open and close the hand. All this will be controlled by a single arduino.

# Motivation
I have always wanted to build a robotic arm. It was basically a longterm goal. I never dared to actually build it though since I was a little bit afraid of my actual abilities. After watching many videos about it, I figured out that I could also build a simple one. And now here I am, designing all the components, which is actually not as difficult as I thought.
